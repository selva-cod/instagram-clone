Instagram Clone

A responsive Instagram Clone built using React, JavaScript, HTML, and CSS.

## Features
- Stories Section
- Posts Feed
- Suggestions Panel
- Navigation
- Profile View
- Responsive Design

## Technologies Used
- React
- JavaScript
- HTML
- CSS
- Vite

## Getting Started

1. Clone the repository:
   git clone https://github.com/selva-cod/instagram-clone.git

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev
